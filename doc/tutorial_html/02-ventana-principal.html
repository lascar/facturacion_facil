🎯 Visión General

La Ventana Principal es el centro de control de Facturación Fácil. Desde aquí puedes acceder a todas las funcionalidades de la aplicación de manera rápida e intuitiva.

🎯 Flujo de Trabajo Típico

📋 Proceso Recomendado

  1. 🏢 Configurar Organización
    • Establecer datos de la empresa
    • Configurar logo y colores
    • Definir configuraciones básicas
  2. 📦 Gestionar Productos
    • Crear catálogo de productos
    • Añadir imágenes y descripciones
    • Establecer precios e IVA
  3. 📊 Controlar Stock
    • Establecer inventario inicial
    • Configurar alertas de stock bajo
    • Monitorear movimientos
  4. 🧾 Crear Facturas
    • Generar facturas profesionales
    • Actualizar stock automáticamente
    • Exportar a PDF

🎨 Personalización de la Interfaz

🌈 Temas de Color

La aplicación soporta diferentes temas de color que puedes configurar desde el módulo de Organización:

  • Claro - Tema por defecto
  • Oscuro - Para uso nocturno
  • Personalizado - Colores de tu empresa

📐 Tamaño de Ventana

La ventana principal es completamente redimensionable:

  • Mínimo - 800x600 píxeles
  • Recomendado - 1200x800 píxeles
  • Pantalla completa - F11

💡 Consejos de Uso

  • 🖱️ Navegación: Usa el ratón para una experiencia más visual

  • 🔄 Actualización: La interfaz se actualiza automáticamente
  • 💾 Guardado: Los cambios se guardan automáticamente
