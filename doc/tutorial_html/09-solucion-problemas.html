🚨 ¿Tienes un problema?

Esta guía te ayudará a identificar y resolver los problemas más comunes que puedes encontrar al usar Facturación Fácil. Sigue los pasos en orden para una resolución efectiva.

🔧 Problemas Comunes y Soluciones

🚫 Problemas de Inicio

❌ La aplicación no inicia

Síntomas: Error al ejecutar, pantalla en blanco, cierre inesperado

Soluciones:

  1. Verificar que Python 3.8+ esté instalado
  2. Reinstalar dependencias: pip install -r requirements.txt
  3. Verificar permisos de escritura en la carpeta
  4. Ejecutar como administrador si es necesario

💾 Problemas de Base de Datos

⚠️ Error de base de datos

Síntomas: "Database locked", datos no se guardan, errores SQL

Soluciones:

  1. Cerrar completamente la aplicación
  2. Verificar que no hay otras instancias ejecutándose
  3. Hacer backup de facturacion.db
  4. Reiniciar la aplicación
📦 Problemas con Productos

🖼️ Imágenes no se muestran

  • ❌ Verificar formato: Solo JPG, PNG, GIF
  • ❌ Tamaño máximo: 5MB por imagen
  • ❌ Ruta válida: Archivo debe existir
  • ✅ Solución: Reseleccionar imagen válida

💰 Problemas con precios

  • ❌ Formato incorrecto: Usar punto decimal (12.50)
  • ❌ Valores negativos: No permitidos
  • ❌ Demasiados decimales: Máximo 2
  • ✅ Solución: Formato: 0.00 a 999999.99
🧾 Problemas con Facturas

📄 PDF no se genera

🔍 Diagnóstico paso a paso:

  1. Verificar datos de organización: Deben estar completos
  2. Comprobar productos: Al menos un producto válido
  3. Verificar permisos: Carpeta de destino escribible
  4. Espacio en disco: Suficiente para generar PDF

🧮 Cálculos incorrectos

✅ Verificaciones:

  • IVA configurado: Porcentaje correcto por producto
  • Cantidades: Números enteros positivos
  • Descuentos: Entre 0% y 100%
  • Redondeo: Automático a 2 decimales
🔍 Problemas de Rendimiento

⚡ La aplicación va lenta

Causas comunes y soluciones:

🗄️ Base de datos grande

  • Hacer limpieza de datos antiguos
  • Optimizar búsquedas con filtros
  • Cerrar ventanas no utilizadas

💻 Recursos del sistema

  • Cerrar otras aplicaciones pesadas
  • Verificar memoria RAM disponible
  • Reiniciar la aplicación periódicamente

🔄 Mantenimiento Preventivo

✅ Consejos para evitar problemas

📅 Rutina semanal

  • Hacer backup de la base de datos
  • Limpiar archivos temporales
  • Verificar actualizaciones
  • Revisar logs de errores

🛡️ Buenas prácticas

  • No cerrar bruscamente la aplicación
  • Mantener datos organizados
  • Usar nombres descriptivos
  • Validar datos antes de guardar
